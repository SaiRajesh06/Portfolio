Real time Prediction of Amine Emission

This project addresses the challenge of predicting amine emissions in carbon capture systems using advanced machine learning techniques. A deep learning approach with an LSTM autoencoder model was employed, leveraging its suitability for time-series data analysis. The model consists of an encoder, decoder, and dense layers to predict amine emissions. Data preprocessing included correlation-based feature selection and normalization on bi-minute data over two weeks. Preliminary results show a mean absolute percentage error below 10%, indicating high prediction accuracy. Future work will explore other architectures like CNN-LSTM hybrids and attentional mechanisms to enhance accuracy further. This research demonstrates the potential of deep learning in forecasting emissions in carbon capture systems.

Predicting Workers Cyber-Resilience Through Mental Health Risk Modeling in Tech Industries

This research project investigates the impact of mental health issues, such as anxiety, depression, and fatigue, on employees' productivity, decision-making, and job performance, and how these factors can increase cybersecurity risks. It employs machine learning algorithms like logistic regression, random forest classifiers, and nearest neighbors and also explores the connections between mental health symptoms and increased cybersecurity vulnerabilities, aiming to motivate tech companies to prioritize mental well-being and data security measures.

Automated Irrigation System

The Automated Irrigation System project addresses water scarcity and labor-intensive irrigation in agriculture by using soil moisture sensors and a microcontroller to automate water delivery. This system conserves water, ensures efficient irrigation, saves farmers' time and labor, and can be powered by reliable sources like batteries or solar panels. Benefits include significant water savings, increased crop yield, reduced manual labor, and scalability to suit various farm sizes. The project aims to enhance agricultural sustainability and efficiency by minimizing manual intervention and optimizing water usage.

DC Motor Speed Control using 8051 Microcontroller

The "DC Motor Speed Control using 8051 Microcontroller" project focuses on regulating the speed of a DC motor through a motor driver circuit connected to an 8051 microcontroller. Hardware setup involves building the motor driver circuit and linking it to the microcontroller. Software implementation includes setting up a timer to generate a PWM signal, calculating timer settings for the desired PWM duty cycle and frequency, and implementing a speed control algorithm to adjust the PWM duty cycle. This project demonstrates precise motor speed control using embedded systems.
